# ⚓잠든 쇳소리를 깨울 고동 (Project Yeongdo)

### 부산 영도 조선 산업의 역사를 기록하고, 데이터로 미래의 맥박을 진단하는 아카이브 프로젝트



#### 🚢 프로젝트 개요

본 프로젝트는 대한민국 근대 조선업의 발상지인 부산 영도를 대상으로 합니다. 한때 국가 경제의 심장이었으나 현재 인구 소멸 위기를 겪고 있는 영도의 상황을 빅데이터 분석을 통해 진단하고, 이를 기반으로 도시 재생의 비전을 제시하는 디지털 아카이빙 작업입니다.

팀명: 잠든 쇳소리 (D조)

팀원: 김은서(역사교육), 정유빈(경영), 정상호(관광컨벤션)

주요 기술 스택: Python (KoNLPy), Chart.js, Tailwind CSS, HTML5


---


#### 📅 프로젝트 여정 (Sections)

## 01. 기획 배경 (Planning)

과거: '깡깡이 아지매'로 대표되는 헌신적 노동과 근대 조선업의 영광.

현상: 지난 30년간 인구 급감 및 1,000가구 이상의 빈집 발생. 전국 83개 소멸위험지역 지정.

연구 방법론:

유튜브(KBS, EBS 다큐 등), 해수부 정책 영상, 부산시 보도자료 데이터 수집.

Python KoNLPy를 활용한 형태소 분석 및 데이터 전처리.

---

## 02. 데이터 분석 결과 (Analysis)

대중의 인식을 과거, 현재, 미래의 타임라인으로 분석한 결과입니다.

과거 (Past): 74.5% 중립/긍정. 쇳소리를 소음이 아닌 '삶의 활기'로 기억.

현재 (Present): 39.2% 부정. '소멸', '폐허', '위험' 키워드 급증.

미래 (Future): 47.4% 긍정. 변화의 계기(MRO 산업, 북극항로 등)가 주어진다면 지지하겠다는 기대감 확인.

---

## 03. 결과물 (Outcomes)

📖 디지털 스토리북

데이터 분석 결과와 영도 조선소의 감성적 이미지를 결합한 인터랙티브 스토리북입니다.

주요 내용: 깡깡이 아지매의 시대 / 노인과 바다 / 다시 뛰는 심장(MRO)

스토리북 바로가기

🎥 다큐멘터리: 잠든 쇳소리

과거의 소음이 미래의 리듬이 되기까지의 과정을 담은 영상 기록입니다. (Running Time 12:45)

영상 시청하기 (Google Drive)

🛠 주요 기능

Responsive Web: Tailwind CSS를 활용한 모바일/데스크탑 최적화 랜딩페이지.

Data Visualization: Chart.js를 이용한 감성 분석 결과 시각화.

Interactive UI: 원페이지 라우팅 기능을 통한 유연한 콘텐츠 탐색.
